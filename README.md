# BIT Música API

API creada con [expressjs](https://expressjs.com/) para la aplicación BIT Música

## Pasos de instalación y ejecución

```shell
git clone https://github.com/jefraroce/bit-musica-api.git
cd bit-musica-api
npm install
node index.js
```
